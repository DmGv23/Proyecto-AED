# Proyecto AED - Sparse Spreadsheet

Implementación de una hoja de cálculo simplificada usando una matriz dispersa enlazada cruzada en C++.

## Funcionalidades

* Insertar y consultar celdas
* Eliminar celdas, filas y columnas
* Suma por filas y columnas
* Promedio, máximo y mínimo
* Impresión de celdas ocupadas

## Tecnologías

* C++
* CMake
* CLion

## Estructura utilizada

Cross-Linked Sparse Matrix con cabeceras dinámicas activas.
